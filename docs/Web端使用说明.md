# Web端使用说明

## 最新 Web 入口

- 一键启动文件：`D:\Codex\全球时间对照\启动Web端.bat`
- 浏览器地址：`http://127.0.0.1:4173`

## 正确打开方式

不要直接双击这些文件：

- `index.html`
- `desktop.html`

这两个是项目入口文件，不是给你直接离线双击使用的页面。

## 推荐方式

1. 双击 `启动Web端.bat`
2. 等待自动打开浏览器
3. 如果没有自动打开，就手动访问 `http://127.0.0.1:4173`

## 关闭方式

- 双击 `关闭Web端.bat`

## 当前定位

- Web端：城市浏览与查询系统
- EXE端：桌面开发工作台

当前阶段先以 Web 端为主继续开发。
